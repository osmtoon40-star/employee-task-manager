# Employee Task Manager

Enterprise task and employee management workspace.

## Projects

- `frontend/` - Vite, React, TypeScript UI.
- `backend/` - Standalone Node.js, Express, MongoDB REST API.

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```

## Run Backend

```bash
cd backend
npm install
npm run dev
```

Backend configuration is read from `backend/.env`.
